# 实验室物资管理工具

这是一个基于 React + Vite 搭建的实验室试剂/耗材管理前端原型，用于解决课题组物资信息分散、重复购买、常用耗材补充不及时等问题。

## 已实现功能
- 试剂柜 / 耗材柜分类入口
- 试剂信息录入与展示
- 耗材信息录入与展示
- localStorage 本地数据保存
- 关键词搜索
- 单条记录删除
- CSV 导出

## 项目背景
课题组人员较多，不同课题购买的试剂和耗材信息分散，传统冰箱贴纸或人工记录流动性差，容易导致重复购买、查找困难或常用物资补充不及时。本项目尝试用轻量化前端工具降低录入成本，提高物资查询和管理效率。

## 技术栈
React / Vite / React Router / localStorage / GitHub Pages

## 当前不足
- 目前仅为前端原型，数据保存在本地浏览器中，尚未接入后端数据库
- 暂不支持多人协同和权限管理
- 暂未实现有效期提醒和低库存提醒

## 后续迭代计划
- 增加试剂有效期提醒，区分“即将过期”和“已过期”
- 增加低库存提醒，辅助常用试剂和耗材补充
- 优化 CSV 导出字段和筛选导出逻辑
- 后续可接入后端数据库，实现多人协同维护
