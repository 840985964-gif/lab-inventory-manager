# 实验室物资管理工具

这是一个基于 React + Vite 搭建的实验室试剂/耗材管理前端原型，用于解决课题组物资信息分散、重复购买、过期试剂不易发现、常用耗材补充不及时等问题。

## 功能
- 试剂/耗材分类入口
- 试剂信息录入与展示
- 耗材信息录入与展示
- 本地数据保存
- 物资搜索
- CSV 导出
- 有效期/低库存提醒

## 项目背景
课题组人员较多，不同课题购买的试剂和耗材信息不透明，传统冰箱贴纸或人工记录流动性差。本项目尝试用轻量化前端工具降低录入成本，提高物资查询和管理效率。

## 技术栈
React / Vite / localStorage / GitHub Pages

## 当前不足
目前仅为前端原型，数据保存在本地浏览器中，尚未接入后端数据库和多人权限管理。
